Conformal Array Antenna Patterns
	This folder contains simulated antenna array patterns for conformal antenna arrays.
	Conical contains eight antenna patterns in .dat format, at 6GHz.
	StackedPatch contains eight antenna patterns in .txt format, eight at 3GHz, and eight at 6GHz.
